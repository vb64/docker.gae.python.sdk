The docker for bitbucket pipelines. Contains:

- Python 2.7.15
- Google App Engine SDK 1.9.54
- requests 2.22.0
- flask 1.0.2
- Babel 2.6.0
- mock 3.0.5
- pylint 1.9.4
- coverage
- flake8
- some tools for autotest GAE code
